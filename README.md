# zvpn
Peer-to-Peer VPN built with [AppNet.link](https://github.com/InstantWebP2P/appnet.link) protocol

### Features

* Smart L2/L3 congestion control
* Support P2P/P2MP/Mesh networking topology
* End-to-End security
* User privacy protection
